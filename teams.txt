type Q or q if you want to get out of the reader

 two qualities to keep in mind when making a commit:
1) The size of your commit
2) The message of your commit

You can only use git diff to compare changes that haven't been staged.
    git diff
The -A flag will add all changes to the staging area, including new files
   git add -A
